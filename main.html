ALP PASSWORD GENERATOR

English

The ALP Random Password Generator allows you to create strong and random passwords. You can use it to enhance password security and protect your accounts across various platforms. To create a password, follow these steps:

  1. Select a hash algorithm from the "Choose the hash algorithm" section.
  2. Enter the desired password length and select a type from the "Choose password type" section.
  3. Click the "Generate" button to create a random password.
  4. You will see the hashed version of the generated password and usage instructions.

To run the bash file, follow these steps in the terminal:

1. Open the terminal.
2. Enter the command `bash alppg.sh` and press Enter.

Türkçe

ALP Rasgele Şifre Oluşturucu, güçlü ve rasgele şifreler oluşturmanıza olanak tanır. Şifre güvenliğinizi artırmak ve hesaplarınızı çeşitli platformlarda korumak için kullanabilirsiniz. Bir şifre oluşturmak için şu adımları izleyin:

  1. "Hash algoritmasını seçin" bölümünden bir algoritma seçin.
  2. Parola uzunluğunu girin ve "Şifre türünü seçin" bölümünden bir tür seçin.
  3. "Oluştur" düğmesine tıklayarak rastgele bir parola oluşturun.
  4. Oluşturulan parolanın karma değerini ve kullanım talimatlarını göreceksiniz.

Bash dosyasını çalıştırmak için terminalde aşağıdaki adımları takip edin:

1. Terminali açın.
2. `bash alppg.sh` komutunu girin ve Enter tuşuna basın.



Parola güvenliği için oluşturulan parolayı saklayın ve kimseyle paylaşmayın.
For password security, keep the generated password safe and do not share it with anyone.
